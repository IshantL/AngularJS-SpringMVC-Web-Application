1.Promocode Generation:-

In this module we are going to facilitate the Event Manager in generation of Promo code, that is only for the paid event after ticket generation. Event manager can generate different promo codes based on the ticket name, which will result in the auto-generation of the promo id. Event Manager will be able to view/update/delete promo codes for the event, while customer can only view it.


2.Attendee Management

The main users of this module are Customer and Event Manager. A registered customer can view all the live events and the tickets details of the event. He can book ticket for himself and other attendees. Event Managers can control how many attendees can register for an event based on Tickets purchased per order .


3.Feedback Management

After attending the event, the attendee can give their feedback using the feedback form. The feedback form will get enabled only after the event gets over. It will get expired within 3 days after activation. Event manager can also view the feedback
